# recruit-information-chatbot

프로젝트 소개
구인공고 안내 챗봇은 잡코리아, 사람인 등 다양한 구인 플랫폼에서 수집한 데이터를 바탕으로,직무 관련 질의에 대한 정확하고 신속한 답변과 함께 맞춤형 구인 공고를 제공하는 챗봇입니다.

![image](https://github.com/user-attachments/assets/48eb4a96-7409-4c7f-8f58-afd2503b7e72)

현재 지원하는 직무는 다음과 같습니다.
1. 데이터 분석가
2. 데이터 엔지니어
3. AI 개발자
4. 챗봇 개발자
5. 클라우드 엔지니어
6. API 개발자
7. 머신러닝 엔지니어
8. 데이터 사이언티스트

주요 목표
구직자 편의성 증대: 복잡한 구인 정보를 간단한 대화를 통해 빠르게 검색하고, 자신에게 맞는 일자리를 찾도록 돕습니다.
기업 인지도 향상: 기업은 챗봇을 통해 자사의 채용 정보를 더 많은 구직자에게 효과적으로 노출할 수 있습니다.

주요 기능
자연어 처리 기반 질의응답: RAG, LangChain, GPT-4를 활용하여 사용자의 다양한 표현 방식을 이해하고, 복잡한 질의에도 정확하게 응답합니다.
맞춤형 구인 공고 추천: 사용자의 경력, 선호하는 기술 스택, 근무지 등 다양한 조건을 고려하여 가장 적합한 구인 공고를 추천합니다.
실시간 데이터 업데이트: 구인 정보를 업데이트하여 당일의 정보를 제공합니다.
다양한 직무 지원: 데이터 분석, 개발, 디자인 등 다양한 IT 분야의 구인 정보를 제공합니다.

기술 스택
Python: 핵심 프로그래밍 언어
Selenium: 웹 크롤링
LangChain: 대규모 언어 모델과 다른 데이터 소스 연결
OpenAI GPT-4: 자연어 처리
streamlit: 웹 애플리케이션 개발
